# Adapti

Adapti is a Single Page Application (SPA) designed to help students manage their academic workload. It automatically prioritises tasks based on deadline urgency and importance rating, so the most critical work always appears first. The app is built using only HTML, CSS, and JavaScript, with `localStorage` used to save user data without needing a server or database.

## Features

- Priority-sorted task dashboard — tasks ranked by importance × urgency score
- Monthly deadline calendar with task names shown on deadline days
- Pomodoro timer with 25, 15, 10 and 5 minute presets
- Progress bar showing overall completion percentage
- JSON import and export for data backup and restore
- XSS prevention — all user input escaped before DOM insertion

## Technologies Used

- HTML
- CSS
- JavaScript
- localStorage

## How to Run

Open `index.html` in a browser.

No server, database, or installation is required.
